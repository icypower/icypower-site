# icypower-site

The public marketing website for **ICYPOWERR** — ice bath, cold exposure &
breathwork experiences in Tel Aviv.

**This is the canonical, only copy of the marketing site.** It deploys to
the Cloudflare Pages project `icypower`, connected directly via GitHub —
pushing to `main` triggers an automatic build, no manual deploy step
needed. Plain static HTML/CSS/JS, no build step of its own.

Do not create another copy of this site in a second repo — if a backup
copy exists elsewhere, it should be a clearly-labeled, automatically
updated mirror (see its own README), never a second place anyone edits.

For the business's other apps (financial tracker, waiver forms) and the
long-term architecture plan, see `icypower/icypower-management`.
